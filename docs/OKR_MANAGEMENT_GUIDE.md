### OKR Types & Use Cases

#### 1. Personal OKRs
**Scope:** Individual development and contributions
**Timeline:** 60-90 days
**Examples:**
- **Objective:** Become proficient in our API integration architecture
- **KR1:** Complete advanced REST API course with 90%+ score
- **KR2:** Design and implement 3 production-ready API integrations
- **KR3:** Deliver internal training session to 10+ engineers

#### 2. Team OKRs
**Scope:** Collective team outcomes
**Timeline:** 90 days standard
**Examples:**
- **Objective:** Achieve 50% faster deployment pipeline by Q2
- **KR1:** Reduce build time from 15 min to 7.5 min average
- **KR2:** Zero failed deployments for 4 consecutive weeks
- **KR3:** Cut code review time from 2 days to 4 hours average

#### 3. Cross-Team OKRs
**Scope:** Multi-team collaboration
**Timeline:** 90-120 days
**Examples:**

- **KR1:** Reduce signup-to-activation time from 3 days to same-day
- **KR2:** Achieve 95% customer satisfaction in onboarding survey
- **KR3:** Cut support tickets by 50% through better self-service

#### 4. Company OKRs
**Scope:** Annual/quarterly strategic goals
**Timeline:** Annual with quarterly check-ins
**Examples:**
- **Objective:** Become the #1 CRM solution in mid-market segment
- **KR1:** Achieve 150% growth in enterprise customer acquisition
- **KR2:** Maintain 95%+ customer retention rate
- **KR3:** Launch mobile-first experience for 10,000+ users

#### OKR Writing Framework

**Objective Checklist:**
- [ ] Is it ambitious but achievable?
- [ ] Can someone unfamiliar understand it?
- [ ] Does it inspire action?
- [ ] Does it fit the evaluation period?

**Key Results Checklist:**
- [ ] Are they specific and measurable?
- [ ] Do they objectively indicate success?
- [ ] Are they time-bound?
- [ ] Is each independent of others?

### Troubleshooting Common Issues

| Issue | Solution | Prevention |
|-------|----------|------------|
| **Too Many OKRs** | Limit to 3-5 objectives max | Use focus technique: what's most critical? |
| **Unrealistic Targets** | Use historical data to set ambitious but achievable goals | Benchmark against past performance |
| **Falling Behind Schedule** | Weekly check-ins and mid-quarter adjustments | Set weekly milestones and checkpoints |
| **Team Misalignment** | Regular sync meetings and shared metrics dashboard | Create team-level OKRs with shared key results |
